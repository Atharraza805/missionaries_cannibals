⚔️ HOW TO PLAY & RULES ▼

The Challenge

Get every person from the left bank to the right bank. The boat carries up to its configured capacity but at least 1 person must row it.

The Sacred Rules

  • 🧑‍⚕️ Missionaries must never be outnumbered on either bank — or they get eaten!
  • ⛵ The boat cannot cross empty — someone must paddle it.
  • Both banks are checked after every move.
  • Exception: A bank with zero missionaries is always safe, even with many cannibals.

Using the + / − Buttons

The Setup panel at the top lets you change the puzzle difficulty:

  • + Missionaries — add another missionary to the left bank
  • − Missionaries — remove a missionary (min 1)
  • + / − Cannibals — same for cannibals (min 1, max 6)
  • + / − Boat — change how many people fit in the boat per trip

Hit Apply & Start New Game to begin with your chosen numbers.

🤖 BFS ALGORITHM EXPLAINED ▼

What is BFS?

Breadth-First Search explores every reachable game state level-by-level, guaranteeing the shortest (fewest moves) solution.

State Space

State = ( leftM, leftC, boatSide ) rightM = totalM − leftM rightC = totalC − leftC Start : (3, 3, LEFT) → Goal : (0, 0, RIGHT) Queue: [(3,3,L,path=[])] Pop (3,3,L) → try all valid boat loads (3,3,L) + send 1M → new state (2,3,R) — SAFE ✓ enqueue (3,3,L) + send 2M → new state (1,3,R) — UNSAFE ✗ skip (3,3,L) + send 1C → new state (3,2,R) — SAFE ✓ enqueue ... First time goal (0,0,R) is dequeued = optimal path!

Why It Works

  • Completeness: Finds a solution if one exists
  • Optimality: Guarantees minimum number of moves
  • States explored: (M+1)×(C+1)×2 — e.g. 32 for classic 3,3
